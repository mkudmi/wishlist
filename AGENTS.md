# AGENTS.md

## Dev Server
- Frontend dev server: `npm run dev`
- Full stack (frontend + backend): `npm run dev:all`

## Working Rules
- Не анализируем весь проект без необходимости; работаем только с релевантными файлами.
- Перед реализацией проверяем, нет ли уже готового компонента/решения для переиспользования.
- Предпочитаем минимальные изменения кода вместо переписывания целых файлов.
- Если изменение минимальное и относится к дизайну (а не к логике), сборку (`npm run build`) не запускаем.
- Если изменение большое и затрагивает логику, сначала проверяем наличие готовых решений в интернете и предлагаем их на выбор перед реализацией.
- Для изменений фронтенда используем frontend skill по умолчанию.
- Если вносим изменения для desktop/ПК, обязательно проверяем и мобильную версию этого же компонента.
- Придерживаемся модульной архитектуры проекта; не смешиваем ответственность компонентов и модулей.
- Кодировка файлов: UTF-8.
- Если работаем на Windows/PowerShell, избегаем bash-специфичного синтаксиса и заранее учитываем ограничения PowerShell, чтобы не допускать командных ошибок.
- Основной язык интерфейса — кириллица; сохраняем корректные русскоязычные тексты и читаемость.
- Перед завершением задачи выполняем финальную проверку изменений на адекватность (логика, стили, регрессии по контексту задачи).
- Перед коммитом обязательно проверяем изменённый код на мусор, дублирование и потенциальные баги.
